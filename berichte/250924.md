# Grundlagen der Linux Shell

Shell = textorientierte Schittstelle zu einem Betriebssystem (Kernel)

Anzeige in der Konsole: Prompt
Eingabe nach dem Prompt

Beispiel Befehl 'date'

``` sh
$ date
```

Ausgabe:

```
Wed Sep 24 07:18:41 AM UTC 2025
```

Befehle können Optionen hben, diese werden mit `_` angegeben.

z.B.:

``` sh
$ date -I
```

Ausgabe:

```
2025-09-24
```

Optioen aus einem Buchstaben = shorthand
Optionen aus mehreren Buchstaben = longhand

longhand mit 2x `--`:

``` sh
$ date --iso-8601
```

Mehr Informationen zu einem Befehl:

- KI
- häufig die Option `--help`

``` sh
$ date --help
```

- man pages

``` sh
$ man date
```

Funktioniert nicht in Replit(derzeit) aber man kann mit `man date` im Internet suchen.
