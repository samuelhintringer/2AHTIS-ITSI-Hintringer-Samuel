## Pipes
Mit pipes kann man in linux den stdin und stdout dynamisch zwischen verschiedenen aufeinanderfolgenden Befehle übergeben.:

## curl ( für Client URL )

Mit curl kann man Dateien von einem Link in stdin oder herunterladen um es weiter zu verarbeiten

```sh
~/workspace$ curl -O https://www.franzmatejka.at/htl/doc/ITSI_2_linux/testdata/klassenkassa.csv
```

```sh
~/workspace$ curl https://www.franzmatejka.at/htl/doc/ITSI_2_linux/testdata/klassenkassa.csv | head | grep Werkstatt | head -n 2
```

## tr ( für Translate )

Mit tr kann man zeichen des stdin modifizieren (verändern, löschen).

### mit pipes

```sh
~/workspace$ echo blablabla | tr bl a
```

## shuf ( für shuffle )

Mit shuf kann man den stdout in zufälliger Reihenfolge ausgeben lassen.

### mit pipes

```sh
~/workspace$ head klassenkassa.csv | shuf
```
