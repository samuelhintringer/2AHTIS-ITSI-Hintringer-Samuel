### Date ISO 8601
Aufgabe:

Recherchiere in der manpage von date wie folgende ISO 8601 Datumsausgabe erzeugt werden kann.
`2023-11-07T08:43:53+00:00`

```sh
date --iso-8601=ns
```

Ausgabe:
```
2025-10-22T06:25:13,743768053+00:00
```

## Directories and Files
- ### Benutzte Befehle in Reihenfolge:
  - mkdir abcd
  - cd abcd
  - mkdir first_dir
  - touch first_dir/abcd.01.1.txt
  - touch first_dir/abcd.01.2.txt
  - touch first_dir/abcd.01.3.txt
  - mkdir second_dir
  - touch second_dir/xyz.02.1.txt
  - touch second_dir/xyz.02.2.txt
  - touch second_dir/xyz.02.3.txt
  - cd abcd/second_dir
  - mkdir ../first_dir/in_first_dir
  - touch ../first_dir/in_first_dir/neu.txt
  - tree abcd

Ausgabe:
```
abcd
├── first_dir
│   ├── abcd.01.1.txt
│   ├── abcd.01.2.txt
│   ├── abcd.01.3.txt
│   └── in_first_dir
│       └── neu.txt
└── second_dir
    ├── xyz.02.1.txt
    ├── xyz.02.2.txt
    └── xyz.02.3.txt

3 directories, 7 files
```

## Befehle - Bewegen und Kopieren
- Befehle 
  - `rm`     löscht Dateien oder Ordner
  - `rmdir`  löscht (leere) Ordner
  - `cp`     kopiert Dateien oder Ordner
  - `mv`     verschiebt oder bennent Dateien oder Ordner um

## Bewegen und Kopieren
- ### Benutzte Befehle in Reihenfolge
  - cd abcd/first_dir/in_first_dir
  - mv neu.txt umbenannt.txt
  - mv ../../first_dir ../../second_dir
  - tree abcd
