# Unterschied absoluter / relativer Pfad

Ein absoluter Pfad geht immer vom Home Verzeichnis aus. Ein relativer Pfad geht von dem Verzeichnis aus in dem man sich befindet.

## Argumente

Argumente werden genutzt um Befehle genauer / detailreicher auszuführen.

Bsp.: `ls -la`  ...

... listet ***alle Elemente ( -a )*** eines Verzeichnis mit ***größe und Datum ( -l )*** auf.
Argumente gibt es in ***Kurzform ( Shorthand )*** und ***Langform ( Longhand )***.
- Longhand: ls --format-long
- Shorthand: ls -l

## History

Die History ist praktisch ein Speicher der alle eingegebenen Befehle für eine bestimmte Zeit speichert (bis zum Herunterfahren des Linuxsystems / löschen der History). Man kann die History mit dem Befehl `history -c` löschen.

## Wichtige Tastenkürzel

Mit ***Strg + C*** bricht man laufende Vorgänge ab

Mit ***Strg + A*** kommt man an den Anfang eines Befehls in der Shell

Mit ***Strg + E*** kommt man ans Ende eines Befehls in der Shell

## Die Befehle und ihre Ausgabe

- `mkdir` -> make directory, erstellt einen neuen Ordner
- `echo` gibt den darauffolgenden text in der shell aus
- `touch` erstellt neue Datei
- `cd /` wechselt ins Root Verzeichnis
- `cd ~` -> change directory, wechselt zu home Verzeichnis
- `rm` -> remove, löscht Dateien
- `ls ../..` zeigt das vorherige Verzeichnis
- `rmdir` -> remove directory, löscht leere Ordner
- `rm -rf test` löscht einen Ordner und alles in ihm
