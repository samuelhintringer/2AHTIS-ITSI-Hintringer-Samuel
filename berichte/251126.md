## StdIn/Out mit C

## 2.5-6

- Erstelle eine Datei `sum.cpp`
- Schreibe ein C Programm das zuerst eine Anzahl (int n) einliest.
- Dann werden n weitere ganze Zahlen eingelesen und die Summe dieser Zahlen ermittelt.
- Gib am Ende die Summe aus
- Compiliere das Programm mit folgender Kommandozeile: `g++ -o sum sum.cpp`
- Behebe evtl. Compiler-Fehler und starte dann das Programm mit: `./sum`
- Erstelle eine Textdatei mit den Input Daten für das Programm sum
- Rufe nun das Programm auf und leite die Textdatei per Dateiumleitung ins stdin des Programms sum um. Schreibe weiters den Output des Programms in eine Textdatei.

```cpp
#include <stdio.h>

int main(){
  int x = 0;
  int y = 0;
  int n = 0;
  scanf("%d")
  for(int i = 0; i < n; i++){
    scanf("%d", &y);
    x+=y;
  }
}

```

Schreibe das vorhergehende Programm so um damit die Anzahl als erster Wert nicht mehr notwendig ist, d.h. das Ende der Daten automatisch erkannt wird.

Hinweis: Es gibt das Konzept des End of File (EOF). Ist stdin die Tastatur so ist EOF ctrl+d. Der Befehl scanf bietet eine Möglichkeit die EOF Situation abzufragen.

```cpp
#include <stdio.h>

int main(){
  int x = 0;
  int y = 0;
  scanf("%d")
  for(int i = 0; 1; i++){
    scanf("%d", &y);
    x+=y;
  }
}
```

### Benutzte Befehle in Reihenfolge

- `touch sum.cpp`
- `vim sum.cpp`
- `g++ -o sum sum.cpp`
- `./sum`
- `touch in.txt`
- `vim in.txt`
- `cat < in.txt > ./sum > out.txt`
