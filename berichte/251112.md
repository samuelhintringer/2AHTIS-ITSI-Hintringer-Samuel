## cat & echo

`Cat & echo` sind stdin/stdout Tools in Linux. Der `cat` Befehl in linux liest eine Eingabe ein und gibt sie aus entweder in der Konsole oder schreibt sie in Dateien, das funktioniert mit Text input direkt von der Tastatur oder aus Dateien.`echo` macht dasselbe nur er kann nicht den Inhalt von Dateien ausgeben.

### 3. os-release

```sh
cat /etc/os-release
```

Ja, ein Hacker kann damit das Betriebssystem und somit möglicherweise zuvor gewesene Schwachstellen des jeweiligen Betriebssystems verwenden um anzugreifen.

```sh
ls -l /etc/os-release
```
Ausgabe:
```sh
lrwxrwxrwx 1 root root 21 Feb  5  2025 /etc/os-release -> ../usr/lib/os-release
```

`ls -l` einer Datei zeigt die Lese-,Schreib-,und Ausführrechte, sowie den Ersteller und Besitzer der Datei. Zum Schluss wird das Erstelldatum und der Dateiname angezeigt.
