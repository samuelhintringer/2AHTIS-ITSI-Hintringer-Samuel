## Einrichtung von Git in Replit und Github
### Schritte in Github
- anlegen eines neuen Repositories

### Schitte in Replit
- Gehe zu tools & files -> git -> settings
- Verbinde bei connections mit github
- Gib bei remote den link zu deiner repository ein und erstelle die remote
- Überprüfe ob dein Github-account bei "commit author" ausgewählt ist
- Gehe zurück zu git
- Übernimm alle files mit einem Commit in die github repository, gib bei message irgendeine message ein, drücke auf "Stage and Commit all changes" und danach auf push
